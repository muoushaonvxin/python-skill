#### 字典,列表只能存放在内存当中

```python
#!/usr/bin/python
# -*- coding:  utf8  -*-

import json

f = open("user_acc.txt",  "w")

info = {
    "alex":  "123",
    "jack":  "4444",
}

f.write(json.dumps(info))
f.close()
```

> ###### 这个时候,发现已经生成文件了.
> ###### json支持列表,元组,字典等数据类型

#!/usr/bin/python
