OrderedDict在python当中可以为字典当中的数据进行排序,使得字典的数据打印出来不再是无序列的
```python
import collections
d1 = collections.OrderedDict()
print(d1)
print(type(d1))

// 输出结果:
OrderedDict()
<class 'collections.OrderedDict'>
```

```python
import collections

d1 = collections.OrderedDict()
d1['a'] = 'A'
d1['b'] = 'B'
d1['c'] = 'C'
d1['1'] = '1'
d1['2'] = '2'
for k,v in d1.items():
    print(k,v)
  
// 输出结果:
a A
b B
c C
1 1
2 2
```

将字典转换成为json数据的格式
```python
import collections
import json

d1 = collections.OrderedDict()
d1['a'] = 'A'
d1['b'] = 'B'
d1['c'] = 'C'
d1['1'] = '1'
d1['2'] = '2'

print(json.dumps(d1))

// 输出结果:
{"a": "A", "b": "B", "c": "C", "1": "1", "2": "2"}
```
