
用于加密相关的操作， 3.x 里代替了 md5 模块和 sha 模块，主要提供 SHA1， SHA224，SHA256，SHA384，SHA512
MD5 算法

import hashlib

m = hashlib.md5()
m.update(b"Hello")
m.update(b"It's me")
print(m.digest())
m.update(b"It's been a long time since last time we ... ")

print(m.digest())       #   2进制格式 hash
print(len(m.hexdigest()))       # 16 进制格式 hash
